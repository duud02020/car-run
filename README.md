# Car Run 🏎️💨

Jogo estilo *Endless Runner* com estética Cyberpunk/Neon, desenvolvido com HTML5 Canvas e Vanilla JavaScript.

## 🕹️ Como Jogar
- **Mouse / Touch**: Arraste para mover para cima e para baixo.
- **Teclado**: Use as setas `Cima/Baixo` ou as teclas `W/S`.
- **Objetivo**: Desvie dos obstáculos rosa neon e colete o máximo de pontos possível.

## 🚀 Como fazer o Deploy (GitHub Pages)

Para colocar seu jogo online e compartilhar com o mundo, siga estes passos via **GitHub Desktop**:

1. **Commit**: No GitHub Desktop, você verá os arquivos que criamos. Adicione uma descrição (ex: "Initial game release") e clique em **Commit to main**.
2. **Push**: Clique em **Push origin** para enviar os arquivos para o GitHub.
3. **Ativar o Site**:
   - Vá para o seu repositório no site [github.com](https://github.com).
   - Clique em **Settings** (Configurações).
   - No menu lateral esquerdo, clique em **Pages**.
   - Em "Build and deployment", selecione a branch `main` e a pasta `/(root)`.
   - Clique em **Save**.
4. **Pronto!**: Em alguns minutos, um link aparecerá no topo da página. Esse será o link do seu jogo!
